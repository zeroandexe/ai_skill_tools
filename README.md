## qdrant-tool

为 AI Agent 提供长期记忆管理能力，支持记忆存储、语义搜索、文件导入等功能。

**核心能力**：
- 记忆条目的增删改查
- 基于向量相似度的语义搜索
- 文本文件批量导入与自动分块
- 集合管理（创建、列出、删除）

**技术栈**：C++ / Qdrant / Ollama

**文档入口**：[qdrant-tool/README.md](./qdrant-tool/README.md)
